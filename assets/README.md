# Assets

This folder is reserved for future public-safe visual assets.

Allowed future assets:

- Synthetic dashboard mockup
- Review workflow diagram
- Sample review note screenshot
- Quality audit mockup
- Public-safe demo walkthrough

Disallowed assets:

- Real trading signals
- Strategy parameters
- Account data
- API keys/tokens
- Telegram configuration
- PnL
- Order history
- Backtest data
- Private prompts
- Proprietary rules

Any future asset should be reviewed before publication to confirm that it
contains only public-safe material and does not provide financial advice,
trading signals, or automated trading claims.
