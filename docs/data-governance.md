# Data Governance

## Public / Private Boundary

This repository contains only public-safe documentation, synthetic examples, diagrams, and portfolio-safe descriptions.

It does not include private trading system contents, real alerts, production deployment details, proprietary strategy logic, or private system internals.

## No Account Data

No exchange accounts, wallet addresses, balances, positions, orders, order history, PnL, account exports, or portfolio holdings are included.

## No Financial Advice

AI Quant Review System is a review workflow and decision-support concept. It does not provide financial advice, trading recommendations, or instructions to buy, sell, or hold any asset.

## No Real Signals

The public repository does not include real trading signals, real candidate opportunities, real alerts, private watchlists, or strategy outputs.

## No Proprietary Strategy Rules

The repository does not include proprietary rules, filters, thresholds, backtest files, real strategy code, or private system internals.

## No Private Prompts

Private prompts and internal prompt workflows are excluded.

## No API Keys / Tokens

API keys, tokens, credentials, `.env` files, exchange configuration, and Telegram tokens are excluded.

## No Telegram Configuration

Private Telegram configuration, bot tokens, channel settings, alert routing, and production notification details are excluded.

## Synthetic Examples Only

All examples are fictional, synthetic, and for portfolio demonstration only.
